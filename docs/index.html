When Coding Meets Culture

Developing data-driven opinions — a four-week introduction to coding in the humanities.

Learn just enough Python to ask humanistic questions of real text — term frequency, sentiment, topic modeling — and to argue from what you find, while staying honest about what computation reveals and flattens. No prior coding experience needed. Everything launches from this page.
